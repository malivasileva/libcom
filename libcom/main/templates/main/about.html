Libcommend

Этот проект является курсовой работой Васильевой Марии, выполненной в рамках обучения на курсе веб-разработки. Основной целью было создание веб-приложения, которое осуществляет поиск книг с помощью API Google Books.

О проекте

Проект реализован с использованием Python и фреймворка Django. Он предоставляет удобный интерфейс для поиска книг, используя мощное API Google Books. Пользователи могут вводить запросы, а система автоматически показывает результаты, предоставляя информацию о книгах, включая название, автора и описание.

Технологии

Для разработки проекта были использованы следующие технологии:

  • Python
  • Django
  • API Google Books
  • HTML, CSS, JavaScript
